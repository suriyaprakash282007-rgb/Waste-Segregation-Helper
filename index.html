    ⚙️ Technology

    How It Works

    Powered by AWS Rekognition and a smart classification layer.

    1. Upload Image

    User takes/uploads a photo of the waste item via the web interface.

    2. API Gateway

    Request routes through Flask backend → AWS API Gateway securely.

    3. AWS Rekognition

    AI model detects objects with high confidence labels using computer vision.

    4. Classify & Return

    Results are mapped to waste categories and returned with disposal instructions.

    HTML5 / CSS3
    JavaScript
    Python Flask
    AWS Rekognition
    DynamoDB
    Amazon S3
    AWS Lambda

    Connect AWS Rekognition

    To enable real AI classification, add your AWS credentials to backend/.env:

    AWS_ACCESS_KEY_ID=your_key
AWS_SECRET_ACCESS_KEY=your_secret
AWS_REGION=us-east-1
USE_DEMO_MODE=false

    Then restart the Flask backend and the app will use real AWS Rekognition!
